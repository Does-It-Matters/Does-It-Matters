# 버그 없는 곳은 없으니까, <br> 개발도, 삶도 패치 패치!
_"세상을 넓게 바라보고 사람을 깊이있게 이해하자"_ &ensp; 많이 서투르더라도 나의 길을 걸어가자

## 개발 경험 
- JavaFX를 활용하여 싱글 윈도우 멀티 뷰 프레임워크 개발
- 포트 & 어댑터 아키텍처, 모듈, 스프링을 활용하여 유연하게 모듈 교체

## 앞으로의 목표
- 스마트 글래스가 개인화된 비서로 대중화될 것을 대비한 토이 프로젝트
    - 촬영된 사진을 개인 노트북 또는 스마트폰에서 관련된 자료를 찾아 온디바이스 AI로 해설해주는 프로젝트
- 프레임워크 개선
    - JavaFX를 Swing으로 전환하여 프레임워크에 대한 이해도를 높이는 교육 자료로 전환 및 개선
    - Kotlin DSL을 활용하여 개선
